Online Course Registration portal for 6th semester students in CSE Department, Tezpur University.
Programming Language used - PHP, JavaScript, MySQL, HTML, CSS

It contains 5 parts-
1) student login/registration
2) updating student details
3) printing bank challan form
4) choosing subjects for course registartion and updating transaction details of bank.
5) printing filled form and submitting in the department.

current demo url of project: http://tucseregistration.esy.es/
